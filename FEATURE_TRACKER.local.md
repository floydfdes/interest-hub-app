# Planned Features

1. Notifications and received engagement counts: show notifications or counters for follows and post likes once notification endpoints are available.
2. Expanded activity tracking: display report submissions, moderation decisions, and hide/mute/block activity types when the backend records them.
3. Remaining pagination coverage: apply paginated handling to search, bookmarks, or recommendations if those APIs become paginated.
4. Two-factor authentication: add setup, verification, and recovery flows once the required auth endpoints are available.
5. Admin warning management: show warning history and allow issuing warnings once moderation endpoints are available.
6. Post share/repost support: display original-post attribution once sharing endpoints and feed behavior are defined.
7. Private-aware follower and following navigation on public profiles once list navigation is designed.
8. Authorized profile post browsing for other users once a profile-post list endpoint is available.
